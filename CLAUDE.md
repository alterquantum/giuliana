## Project Overview

**Giuliana** is a PHP 8.1 web application running locally via XAMPP (Apache). It is currently in early development — the codebase is a blank slate with no framework, routing, or database yet configured.

- **Local URL**: http://localhost/giuliana/
- **Entry point**: `index.php`
- **PHP version**: 8.1
- **Server**: Apache via XAMPP
- **IDE**: NetBeans (project config in `nbproject/`)

## Running the App

Start XAMPP (Apache) and navigate to http://localhost/giuliana/. There is no build step.
